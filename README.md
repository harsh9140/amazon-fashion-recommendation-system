\# 🛍️ Amazon Fashion Recommendation System



\## Overview



A content-based recommendation system built using Python and Machine Learning that recommends similar Amazon fashion products based on product descriptions.



\## Features



\- Data preprocessing

\- Text cleaning using NLTK

\- Product recommendation

\- Cosine similarity

\- Jupyter Notebook implementation



\## Tech Stack



\- Python

\- Pandas

\- NumPy

\- Scikit-learn

\- NLTK

\- Matplotlib

\- Jupyter Notebook



\## Installation



```bash

git clone https://github.com/harsh9140/recommendation-system.git



pip install -r requirements.txt

```
